## 安装
```
weui: 0.0.5
```

## 如何运行案例
#### 1. clone 代码
```
git clone https://github.com/allan-hx/flutter-weui.git
```

#### 2. 安装依赖
```
flutter packages get
```

#### 3. 进入flutter-weui-example目录 执行以下命令
```
flutter run
```
